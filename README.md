The goal of this repository is to practice and demonstrate Python skills through small, focused projects.

Each project is designed to reinforce concepts such as:

- Python fundamentals
- Control flow and functions
- Data structures
- File handling
- JSON data management
- Command-line applications
- Basic web applications
- Project organization

## Featured Project: Quiz Game Web

The first project in this collection is a web-based quiz game.

Unlike a basic command-line quiz, this version is designed as a simple web application where users can answer randomly selected questions and see their final score.

Planned features:

- Web interface
- Random question selection
- Questions stored in a JSON file
- Score tracking
- Basic result screen
- Simple and clean project structure

## How to Run a Project

Each project has its own folder inside `projects/`.

Example:

    cd projects/01-quiz-game-web

Install dependencies if needed:

    pip install -r requirements.txt

Run the project:

    python app.py

Some projects may have different instructions depending on whether they are CLI apps, web apps, or automation scripts. Each project folder will include its own README when necessary.
